TESTAR VALIDAÇÕES:
==================

1. ERRO: Conta não existe
   - Login com username: ninguém
   - Password: qualquer coisa
   - Toast: "Essa conta não existe."

2. ERRO: Senha inválida
   - Login com username: teste123
   - Password: errado123
   - Toast: "Senha inválida."

3. ERRO: Conta duplicada
   - Tente registrar com mesmo username
   - Toast: "Essa conta já existe."

4. ERRO: Campos vazios
   - Registre sem preencher campos
   - Toast: "Preencha todos os campos."

==================================

2. Vá para: Application → Local Storage → seu-site
3. Em localStorage verá:
   { "accounts": [...], "accountLogs": [...] }

4. Para ver formatado no console:
   console.log(JSON.parse(localStorage.getItem('accounts')))
