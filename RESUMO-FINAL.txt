FLUXO COMPLETO:
===============

1. PRIMEIRA VISITA
   ├─ Registro (index.html → registro.html)
   │  ├─ Preenche: username, senha, sexo
   │  ├─ Seleciona servidor (Servidor 1)
   │  ├─ Aceita termos
   │  └─ Conta salva em localStorage
   │
   ├─ Vai para verificação (process.html)
   │  └─ Toast: "Conta registrada com sucesso!"
   │
   └─ Retorna para login (index.html)

2. FAZER LOGIN
   ├─ Preenche: username, senha
   ├─ Sistema valida em localStorage
   ├─ Toast: "Login bem-sucedido!"
   ├─ Vai para painel (painel.html)
   └─ sessionStorage salva usuário logado

3. ACESSAR MINHA CONTA
   ├─ Clica em "Minha Conta" no menu
   ├─ Abre minha-conta.html
   ├─ Carrega avatar (by sexo)
   ├─ Exibe todas informações
   └─ Pode clicar em "DESCONECTAR"

4. DESCONECTAR
   ├─ Remove sessionStorage['loggedUser']
   ├─ Mantém localStorage['accounts'] intacto
   ├─ Redireciona para login
   └─ Dados da conta continuam salvos

5. FAZER LOGIN NOVAMENTE
   ├─ Digite username e senha
   ├─ Sistema encontra conta em localStorage
   ├─ Valida senha
   ├─ Faz login novamente
   └─ Todos dados estão intactos
